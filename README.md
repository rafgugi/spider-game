# spider-game
Get away spider! Run on Visual Studio with Kinect SDK.

## How do I get set up?
 - Install Visual Studio
 - Install [SDK Kinect 1.8](https://www.microsoft.com/en-us/download/details.aspx?id=40278)

## Start the game
 - Buka Visual Studio
 - File > Open > Project/Solution
 - Pilih `ShapeGame.sln`
